About the author

A web page should always contain some information which enables a reader to find out who wrote it or is responsible for it. This can include an email address (which can be linked), a URI, or even a physical address. The type address is used to identify this information, and it can contain other block-level elements such as paragraphs, headings, and lists.

There is a default style for address elements in most browsers but the author can also use style sheets to provide a suggested presentation.
