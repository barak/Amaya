Creating an Internal Link

To create a link to a target anchor or target element in the same document:

  1. Select to select (by click and drag or by keyboard) the text for the link you are creating.

  2. Click the Link button (first case) or select the entry "Create or change link" in the Links menu (second case).
    • In the first case, the cursor changes from an arrow to a hand to let you click the target icon (or an anchor with name).
      • If the target is displayed (you may scroll to display it), click it to create the link.
      • If you press the F2 or Delete key, or click a part of the document which cannot be a valid target, a dialog prompts you for the location of the target. Type the URI of the target and then Confirm to create the link.
    • In the second case, a dialog prompts you for the location of the target.
      • If the target is displayed (you may scroll to display it), click the Click button then click the target icon to create the link.
      • Else type the URI of the target and then Confirm to create the link.
