Copying Attributes

An attribute is not the same thing as element and an attribute cannot be copied by the Copy or Paste options on the Edit menu.

To copy an attribute, you must use the Attributes menu:

  1. Select the element with the attribute you want to copy.

  2. Select the appropriate attribute entry from the Attributes menu.

  3. Select the new element to which you want to associate the current attribute and click the Apply button.
