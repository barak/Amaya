Resizing Image Map Areas

After you have defined an area within an image map, you may need to resize that area.

To resize the image map area directly from the keyboard

  1. Move the cursor into an area of the image map.
  2. Press the Control key and right-click the mouse. This will highlight the image map area.
  3. Release the Control key but continue to hold down the right-button of the mouse. Use the mouse to drag the image area to the size you want.

To resize the area using the Structure view

  1. Select Show Structure from the Select Show Structure menu (or press Alt + v then Alt + s). The structure view of the document will open in another window.
  2. Edit the coords attribute manually. For example, coords=25,29,16 might become coords=25,28,24.
  3. The image map area will resize in the open browser window.
