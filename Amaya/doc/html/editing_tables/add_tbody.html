Adding a New Table body

To create a new tbody:

  1. To create a new table body above an existing table body, click at the beginning of the first cell in the first row of the table body and select the tbody entry in the Table sub-menu.
  2. To create a new table body below an existing table body, move the cursor to the end of the last cell in the last row of the table body and select the tbody entry in the Table sub-menu.

New table bodies can also be easily added to a table using the Enter key. When a whole table body is selected (use F2), pressing the Enter key create a new table body after the selected table body.
