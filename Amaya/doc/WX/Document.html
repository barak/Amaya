Creating new Documents

Amaya allows you to create both local or remote documents. Four types of documents are available:

  • XHTML
  • MathML
  • SVG
  • CSS

To create a document:

  1. Choose New from the File menu, and then select the file type that you want to create.
  2. At the prompt, type the URI or the local path name of the new document (a default name is assigned to the document if no name is specified). You can use the file browser to find a local directory.
  3. Choose the location of the new document : current tab, new tab or new window
  4. Click Open. A new document opens, and a root element is created.

To detach a document from a template

  1. Choose New from the File menu, and then select From template.
  2. Amaya asks you to choose a template. You can select one from the list or move to another directory where the template is located.
  3. Under the label Save document as..., choose the location where you want Amaya to save your new document. You can use the file browser to choose a location.
  4. Choose the location of the new document : current tab, new tab or new window
  5. Click Open. The template is copied, and links are updated.

Editing a document detached from a template

The template specifies which areas are locked and what you can change in the document. A dashed red frame is drawn around editable areas.
