librdf_model_context_add_statement

Name

librdf_model_context_add_statement --  Add a statement to a model with a context

Synopsis

int librdf_model_context_add_statement (librdf_model* model, librdf_node* context, librdf_statement* statement);

Arguments

model

librdf_model object

context

librdf_node context

statement

librdf_statement statement object

Description

It must be a complete statement - all of subject, predicate, object parts must be present.

If context is NULL, this is equivalent to librdf_model_add_statement

Return value

Non 0 on failure
