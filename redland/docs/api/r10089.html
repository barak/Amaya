Synopsis

int librdf_storage_has_arc_in (librdf_storage * storage, librdf_node * node, librdf_node * property);

Arguments

storage

librdf_storage object

node

librdf_node resource node

property

librdf_node property node

Return value

non 0 if arc property does point to the resource node
