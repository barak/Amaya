librdf_storage_set_feature

Name

librdf_storage_set_feature --  set the value of a storage feature

Synopsis

int librdf_storage_set_feature (librdf_storage* storage, librdf_uri* feature, librdf_node* value);

Arguments

storage

librdf_storage object

feature

librdf_uri feature property

value

librdf_node feature property value

Return value

non 0 on failure (negative if no such feature)
