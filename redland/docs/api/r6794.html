librdf_hash_put_strings

Name

librdf_hash_put_strings --  Insert key/value pairs into the hash as strings

Synopsis

Arguments

hash

hash object

key

key

value

value

Description

The key and values are copied into the hash, no sharing i s done.

non 0 on failure
