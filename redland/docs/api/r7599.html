librdf_hash_memory_open

Name

Synopsis

Arguments

context

memory hash context

identifier

identifier - not used

mode

access mode - not used

is_writable

is hash writable? - not used

is_new

is hash new? - not used

options

librdf_hash of options - not used

Return value

non 0 on failure
