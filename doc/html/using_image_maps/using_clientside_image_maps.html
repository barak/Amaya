Using client-side image maps

Amaya enables you to use client-side image maps.

To activate an area in an image map

  • Double-click on the area within the image. Amaya retieves the URI associated to this area, and opens it in the browser.

To display the existing areas within an image map

  • Select Show Map Areas from the Views menu.

    Note:   You can also display existing areas directly from the keyboard Ctrl u Ctrl m.
