Editing Attributes in the Structure View

All attributes are displayed in the Structure view. Attributes may be displayed in blue or pink.

  • Attributes whose value can be chosen freely are displayed in pink. You can edit the value like any other character string.

  • When the attribute value is displayed in blue, you can only change it with the Attribute menu. This ensures that only one of the correct values is chosen.
